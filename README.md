# TaskFlow

A Trello-style project management platform, engineered the way a real product team would build it.

> Phase 1 in progress — see [docs/](docs/) for the full planning.

## Structure
- `apps/api` — Spring Boot 3.5 / Java 21 backend (modular monolith)
- `apps/web` — React 18 + TypeScript frontend
- `services/` — future extracted microservices
- `docs/` — all project documentation
